What is dotagents

Three things in one.

The App

A unified control plane for your AI agents. Manage profiles, delegate tasks, and watch agents execute tools in real time.

The Protocol

An open standard for agent skills, memories, and commands. Define once in .agents/, works everywhere.

Agent Skills

Reusable capabilities your agents can learn. Portable, shareable, and composable — not locked into any single vendor.

Capabilities

Everything you need.

A complete operating system for AI agents with real tool execution.

orchestration

Agent Profiles

Specialized AI personas with distinct skills. Skill-based delegation, persistent memory, and ACP multi-agent coordination.

mcp tools

Real Execution

Tool execution via Model Context Protocol. OAuth 2.1 auth, real-time progress tracking, and conversation context.

memory

Persistent Context

Agents remember across sessions. Project context, user preferences, and learned patterns stored in the open .agents/ format.

voice + tts

Voice Interface

Hold-to-record in 30+ languages with auto-insert. 50+ AI voices via OpenAI, Groq, and Gemini for spoken output.

observability

Langfuse Tracing

Built-in LLM observability. Trace calls, monitor token usage, and debug agent behavior with Langfuse.

multi-provider

Any Model

Bring your own provider — OpenAI, Groq, or Gemini. Switch models per agent. Rate limit handling built in.

How it works

Three steps.

1

Define your agents

Create agent profiles with specific skills, tools, and memory. Each agent is a specialist in its domain.

2

Agents think & act

Give a task via voice or text. The right agent picks it up, reasons through it, and executes tools in real time.

3

Results delivered

Output flows back to you — inserted into your app, spoken aloud, or displayed. Skills persist for next time.

The .agents protocol

Define once.
Run everywhere.

Skills you create for DotAgents work across every tool that adopts the protocol. Protocol first, product second.
